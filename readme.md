# 🏏 IPL Match Performance Analysis (2008–2024)

## 📊 Overview
This project analyzes IPL match data from 2008 to 2024 to extract key insights about team performance, toss decisions, venues, and player impact.

---

## 🔍 Key Insights
- Most dominant team in IPL history
- Best chasing team
- Toss impact on match outcomes
- Venue-based performance analysis
- Player of the Match trends
- Win streaks and pressure performance

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
